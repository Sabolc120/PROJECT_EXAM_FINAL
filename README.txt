Szoftver elindítása

Backend elindítása-
1. Build (Kalapács ikon a szerkesztőn, IntelliJ) 
2. mvn spring-boot:run
Abban az esetben ha hibába ütközött a backend indításakor, akkor egy másik alternatíva
1. IntelliJ szerkesztőjében válassza ki a CakeExamBackendApplication fájlt, majd pedig a szerkesztőben kattintson a jobbra mutató nyilra, amivel el lehet indítani.
Frontend elindítása-
1. npm install
2. ng-serve
xampp elindítása: Apacha - Start, MySql - Start & Admin.

Frontend elindítása-
1. Miután a projekt megvan nyitva, és sikeresen telepítve lett minden, egyszerűen csak beírjuk a Visual Studio Code termináljába hogy "ng-serve"

Telepítés-
1. Intellij Community és Visual Studio Code letőltése,
2. Node.js letőltése, Angular CLI telepítése,
2.1 Lehetséges hogy kelleni fog egy powershell engedély, a frontend elindításához,
3. Maven letőltése, környezeti változók konfigurálása,
4. Java JDK  letőltése, környezeti változók konfigurálása,
5. XAMPP letőltése, repóban lévő "mycakewebshop" adatbázis használata, és az ibdata bemásolása az jelenlegi MySQL mappába,
5.1 Fontos! Ha az 5. lépést elvégezzük a régi adatbázis adatok eltünhetnek, ezért ha fontos adatok vannak rajta akkor ajánlott egy mentést készíteni,
6. ChromeDriver letőltése a tesztprogram lefuttatásához.
